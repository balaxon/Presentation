Lodash
What is Lodash?
Lodash is a JavaScript library which provides utility functions for common programming tasks using the functional programming paradigm.
Why Lodash?
Lodash makes JavaScript easier by taking the hassle out of working with arrays, numbers, objects, strings, etc.

Lodash is a JavaScript library that helps programmers write more concise and maintainable JavaScript.
It can be broken down into several main areas:
Utilities:
For simplifying common programming tasks such as determining type as well as simplifying math operations.
For exaple:
  • times
  • range

Function:
Simplifying binding, decorating, constraining, throttling, debouncing, currying, and changing the pointer.
For exaple:
  • ary

String:
Conversion functions for performing basic string operations, such as trimming, converting to uppercase, camel case, etc.
For exaple:
  • camelCase
  • lowerCase
  • repeat
Array:
Creating, splitting, combining, modifying, and compressing
For exaple:
  • compact
  • drop
  • head
Collection:
Iterating, sorting, filtering, splitting, and building.
For exaple:
  • countBy

Object:
Accessing, extending, merging, defaults, and transforming.
For exaple:
  • assign

Seq:
Chaining, wrapping, filtering, and testing.
